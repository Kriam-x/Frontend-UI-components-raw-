# Frontend UI Components & Pages

## Overview
Collection of responsive frontend components and pages showcasing practical HTML, CSS, Tailwind, and JavaScript skills. Includes navigation bars, landing pages, login pages, and a pure CSS carousel.

## Features
- Responsive navigation bars
- Landing pages with modern layouts
- Basic login page implementation
- DOM manipulation for dynamic behavior
- Pure CSS carousel component
- Tailwind CSS used for rapid styling

## Tech Stack
- HTML, CSS, JavaScript
- Tailwind CSS
